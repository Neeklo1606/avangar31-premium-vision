3. **Header.jsx** — между лого и кнопкой задать явный gap (например обёртка `flex items-center gap-4` или `gap-3`), при необходимости подправить отступ у лого (`mr-3` / `mr-4`).

| Что | Детали |
|-----|--------|
| **Файл** | `src/components/sections/SearchSection.jsx` |
| **Текущее** | `grid grid-cols-2 lg:grid-cols-5` — равные колонки. |
| **Цель** | **Row 1**: 2 широкие + 3 узкие; **Row 2**: 3 узкие + 2 широкие |

**Шаги:**
1. Реализовать сетку через CSS Grid:
   - Вариант A: две строки с явными колонками, например `grid-cols-5` с `col-span-2` для широких и `col-span-1` для узких в нужных позициях.

2. Точные пропорции: первая строка — [wide, wide, narrow, narrow, narrow]; вторая — [narrow, narrow, narrow, wide, wide]. Например: 5 колонок, row1: span 2, span 2, span 1, span 1, span 1; row2: span 1, span 1, span 1, span 2, span 2.
