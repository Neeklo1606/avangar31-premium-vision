# План: Карточки категорий Zero-блока — приведение к эталону

## Цель
Исправить карточки категорий (Новостройки, Вторичная недвижимость, Аренда и т.д.) для соответствия эталонному макету.

---

## Выполненные изменения

### 1. Вертикальная структура (flex-column)
- **До:** текст сверху, иконка снизу (или текст слева, иконка справа)
- **После:** иконка сверху, текст снизу — как в эталоне
- `flex-direction: column`, `align-items: flex-start`
- Фиксированный gap: `gap-3` (12px) между иконкой и текстом
- Без absolute/negative margins

### 2. Типографика (единый стиль)
- `font-family: Rubik` — без подмен
- `font-weight: 500` (font-medium) — одинаковый для всех
- `font-size: text-sm` (mobile/tablet), `text-base` (desktop) — без auto-scale
- `line-height: 1.35` — фиксированный, без прилипания
- `line-clamp-2` — максимум 2 строки, перенос длинных названий
- `break-words` — длинные слова переносятся

### 3. Контейнер текста
- `max-w-full` — текст не растягивается за пределы карточки
- `text-left` — выравнивание по левому краю
- `min-w-0` — для корректного shrink в flex

### 4. Высота карточек
- `min-h-[88px] lg:min-h-[96px]` — минимальная высота
- `h-full` — заполнение ячейки в grid с `auto-rows-fr`
- Все карточки в ряду имеют одинаковую высоту

### 5. Размеры иконок (без изменений)
- Mobile: `w-16 h-16` (64px)
- Desktop: `lg:w-20 lg:h-20` (80px)
- `object-contain` — без обрезаний

## Критерии проверки
- [ ] Иконка всегда сверху, текст всегда снизу
- [ ] Между иконкой и текстом стабильный отступ (gap-3)
- [ ] Текст не залипает к иконке
- [ ] Одинаковый шрифт для всех карточек
- [ ] Длинные названия (Коммерческая недвижимость) не ломают карточку, max 2 строки
- [ ] На desktop: 1–2 строки, без налезания
- [ ] На mobile: текст не выходит за границы
- [ ] Визуальная плотность как в эталоне
