## Цель
Привести секцию «Новые объявления» (OffersSection) к соответствию эталонному макету.

**Затрагиваемые файлы:**
- `src/components/sections/OffersSection.jsx` — секция, заголовок, сетка, CTA-блок
- `src/components/ui/PropertyCard.jsx` — карточка объекта, бейджи, иконка избранного

## TASK 2.1 — Заголовок блока «Новые объявления»

**Текущее состояние:**
- `text-2xl lg:text-3xl` (20px / 24px)
- `font-semibold` (600)
- `mb-5 lg:mb-6` (20px / 24px)
- Выровнен по левому краю

**Действия:**
- [ ] Сравнить с макетом: font-size, font-weight, line-height, letter-spacing
- [ ] Выровнять заголовок по левому краю карточной сетки (align с первой колонкой)
- [ ] Уменьшить вертикальный отступ сверху секции (`py-8 lg:py-10` → по макету)
- [ ] Уменьшить margin-bottom заголовка, чтобы он визуально «открывал» сетку, а не висел отдельно

**Результат:** Заголовок совпадает с макетом по размеру, жирности и отступам.

## TASK 2.2 — Карточка объекта: радиус и контейнер

**Текущее состояние (PropertyCard):**
- `rounded-lg` (12px)
- `bg-white`
- `border border-gray-light/30`
- `shadow-sm hover:shadow-md`
- Padding контента: `p-3`

**Действия:**
- [ ] Привести border-radius к макету (например, 10px, 12px или 16px)
- [ ] Уточнить background-color (белый / светло-серый)
- [ ] Подкорректировать внутренние padding
- [ ] Убрать ощущение «перегруженности» — возможно, ослабить тень или бордер

**Результат:** Карточки выглядят аккуратно, без лишней визуальной тяжести.

## TASK 2.3 — Изображение в карточке

**Текущее состояние:**
- Контейнер: `overflow-hidden`, без отступа от краёв
- Изображение: `object-cover`, на всю ширину
- Скругление: наследуется от `rounded-lg overflow-hidden` карточки

**Действия:**
- [ ] Задать радиус изображения (внутренний border-radius) по макету
- [ ] Добавить отступ изображения от краёв карточки (margin/padding)
- [ ] Исключить «залипание» изображения к краям

**Результат:** Изображение встроено в карточку, без резких краёв.
